# abbayedesmorts-java

Port of Abbaye Des Morts to Java / LWJGL

## Requirements

* Java 17+

* Maven

## Build

```
mvn clean spotless:apply package
```

## Run

```
java -XstartOnFirstThread -jar target/abbayedesmorts-1.0.0-SNAPSHOT.jar
```

The -XstartOnFirstThread flag is required on Mac
